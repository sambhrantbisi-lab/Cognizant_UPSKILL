# Local Community Event Portal

A single-page front-end project created for Cognizant Digital Nurture 4.0/5.0 Module 1 practice.

## Files

- `index.html` - semantic HTML5 structure, Bootstrap 5 layout, jQuery hooks, and all content sections
- `styles.css` - external stylesheet with responsive CSS3 styling
- `main.js` - JavaScript interactions, validation, storage, fetch example, geolocation, and DOM rendering
- `README.md` - project overview and usage

## Features Covered

This portal demonstrates as many Module 1 exercises as possible:

- HTML5 semantic layout
- Navigation bar with internal page links
- Welcome banner using `id` and `class`
- Image gallery
- Event registration form with validation
- Video section
- Geolocation API
- LocalStorage and SessionStorage
- Character counter in textarea
- Event handling with `onclick`, `onchange`, `onblur`, `ondblclick`, and keyboard events
- CSS3 selectors, gradients, background image, Google Fonts, tables, lists, box model, media queries, and mobile layout
- JavaScript DOM manipulation, arrays, objects, functions, closures, fetch, async/await, error handling, and ES6 syntax
- Bootstrap 5 navbar, grid, cards, forms, buttons, typography utilities, modal, accordion-style content, flexbox utilities, responsive visibility classes, and Bootstrap Icons
- jQuery document ready, click, double click, mouse enter/leave, keypress, hide/show/toggle, fadeIn/fadeOut, and DOM manipulation

## How to Run

1. Open the folder in VS Code.
2. Open `index.html` directly in the browser or use Live Server.
3. If your browser blocks geolocation on `file://`, allow location access or run with Live Server for full API support.

## Notes

- External images, fonts, Bootstrap, jQuery, and icons are loaded from public CDNs.
- The page is designed to work directly without a backend.
- The registration form stores draft data and the latest submission in `localStorage`.
